Interface Zend_Media_Id3_Language

Description
Description | Methods (details)

The Zend_Media_Id3_Language interface implies that the implementing ID3v2 frame supports its content to be given in multiple languages.

The three byte language code is used to describe the language of the frame's content, according to ISO-639-2. The language should be represented in lower case. If the language is not known the string 'und' should be used.
