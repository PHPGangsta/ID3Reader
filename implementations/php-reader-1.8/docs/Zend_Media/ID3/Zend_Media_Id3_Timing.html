Interface Zend_Media_Id3_Timing

Description
Description | Methods (details) Constants (details)

The Zend_Media_Id3_Timing interface implies that the implementing ID3v2 frame contains one or more 32-bit timestamps.

The timestamps are absolute times, meaning that every stamp contains the time from the beginning of the file.
