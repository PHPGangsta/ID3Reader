Zend_Media_Mpeg_Abs_Frame

Description
Description | Vars | Methods (details) Constants

This class represents an MPEG Audio Bit Stream frame as described in ISO/IEC 11172-3 and ISO/IEC 13818-3 standards.

To accommodate fast header processing the error checking data and the audio data are lazy fetch by default. You can change this behaviour by giving a proper option to the Zend_Media_Mpeg_Abs class.

getModeExtension (line 412)

Returns the mode extension used in CHANNEL_JOINT_STEREO mode.

In Layer I and II the return type indicates which subbands are in intensity_stereo. All other subbands are coded in stereo.

  • MODE_SUBBAND_4_TO_31 -- subbands 4-31 in intensity_stereo, bound==4
  • MODE_SUBBAND_8_TO_31 -- subbands 8-31 in intensity_stereo, bound==8
  • MODE_SUBBAND_12_TO_31 -- subbands 12-31 in intensity_stereo, bound==12
  • MODE_SUBBAND_16_TO_31 -- subbands 16-31 in intensity_stereo, bound==16
In Layer III they indicate which type of joint stereo coding method is applied. The frequency ranges over which the intensity_stereo and ms_stereo modes are applied are implicit in the algorithm. Please see MODE_ISOFF_MSSOFF, MODE_ISON_MSSOFF, MODE_ISOFF_MSSON, and MODE_ISON_MSSON.
