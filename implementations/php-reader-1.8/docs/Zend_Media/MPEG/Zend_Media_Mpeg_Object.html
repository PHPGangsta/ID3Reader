		nextStartCode (line 131)
	
 
	
	

Finds and returns the next start code. Start codes are reserved bit  patterns in the video file that do not otherwise occur in the video stream.


All start codes are byte aligned and start with the following byte  sequence: 0x00 0x00 0x01.
