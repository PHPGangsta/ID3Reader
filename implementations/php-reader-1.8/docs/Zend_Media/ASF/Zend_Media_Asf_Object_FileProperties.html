Returns the amount of time to buffer data before starting to play the

BROADCAST = 1 (line 48)

Indicates, if set, that a file is in the process of being created (for

example, for recording applications), and thus that various values stored in the header objects are invalid. It is highly recommended that post-processing be performed to remove this condition at the earliest opportunity.

SEEKABLE = 2 (line 59)

Indicates, if set, that a file is seekable. Note that for files

containing a single audio stream and a Minimum Data Packet Size field equal to the Maximum Data Packet Size field, this flag shall always be set to 1. For files containing a single audio stream and a video stream or mutually exclusive video streams, this flag is only set to 1 if the file contains a matching Simple Index Object for each regular video stream.
