Constructor __construct (line 47)

Constructs the Zend_Io_FileWriter class with given path to the file. By default the file is opened in write mode without altering its content (ie r+b mode if the file exists, and wb mode if not).

  • throws: Zend_Io_Exception if the file cannot be written
